Event Flyers

Event flyers created using Photoshop CS5.5. Client usually gives a basic color scheme or "vibe" with the information they want on the flyer. Challenges I face are when the client wants to include alot of information on the flyer which can compromise or limit the overall design of the flyer.

Business/ Event Banners

Banners created using Photoshop CS5.5. Client usually gives a basic color scheme or theme to work with. Most of the time I have the liberty to design freely. My favorite banners to do are party banners with themes that are uncommon or unique.

Product Mockups

Mockups created using Photoshop CS5.5. Doing mockups is one of my favorite things to do. I get to visually see how my designs would look before actually creating the product. I mostly do themed event mockups for birthdays,babyshowers etc...

Logos

Logos created using Illustrator and Photoshop CS5.5. Clients provide a concept, inspiration and color scheme, which I then implement when designing their logo.
